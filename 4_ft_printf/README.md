# PRINTF

**Project Objectives :** Parsing, variadic functions

## Intro

Discover variadic functions and code optimization with printf function.

### Prototype

`int ft_printf(const char *format, ...);`

### Features
> - Flags : `sSpdDioOuUxXcC%`
> - Conversions : `lhjzL`
> - Formating : `+- #0`
> - length & precision
